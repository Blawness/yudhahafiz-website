# Yudha Hafiz Personal Website

[![CI](https://github.com/Blawness/yudhahafiz-website/actions/workflows/ci.yml/badge.svg)](https://github.com/Blawness/yudhahafiz-website/actions/workflows/ci.yml)

The professional digital home of **Yudha Hafiz** — a Fullstack Developer specializing in Web Applications, Automation, and AI Integration. This website is built for high-performance, SEO optimization, and premium user experience.

## ✨ Features

- **🚀 High Performance**: Built with Next.js 15 and React 19 for lightning-fast delivery.
- **🎨 Premium Design**: Modern, responsive UI using Tailwind CSS v4 and Framer Motion animations.
- **🔍 SEO Optimized**: Dynamic sitemaps, semantic HTML, and JSON-LD structured data.
- **📱 Fully Responsive**: Optimized for all devices, from mobile to ultra-wide displays.
- **📝 Markdown Blog**: Seamless content management using Markdown with metadata support.
- **🤖 AI & Automation Showcase**: Highlighting real-world impact through specialized solutions.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📂 Project Structure

| Path | Description |
|------|-------------|
| `/app` | Next.js App Router pages and global layouts. |
| `/components` | Reusable React components (UI, Layout, Sections, Animations). |
| `/content` | Markdown content for the blog. |
| `/lib` | Utility functions and shared logic. |
| `/public` | Static assets (images, icons, etc). |

## 🌐 Routes

| Route | Description |
|-------|-------------|

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Blawness/yudhahafiz-website.git
   cd yudhahafiz-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🏗️ Development & Quality
